## About CodeForge

CodeForge is a fully configured development environment built for Claude Code. It provides:

- **DevPod Integration** - Containerized development with persistent configuration
- **MCP Servers** - Pre-configured Model Context Protocol servers for enhanced Claude capabilities
- **Modern Tooling** - All the development tools you need, pre-installed and configured
- **Claude-Optimized** - Settings and configurations tuned for AI-assisted development

### Features

- One-click setup for Claude Code development
- Persistent workspace configuration
- Pre-configured MCP servers
- Modern development tools (Node.js, Python, Git, etc.)
- Extensible plugin system

### Getting Started

Clone the repository and open in DevPod or VS Code with Dev Containers extension.
